====== Menu Inicial ======
 1 - Menu de Livros      -> Abre o menu de Livros
 2 - Menu de Usuarios    -> Abre o menu de Usuarios
 3 - Menu de Emprestimos -> Abre o menu de Emprestimos
 4 - Outros              -> Abre as Outras opcoes
-1 - Encerrar            -> Encerra o Programa

=>                       -> Espera a entrada de uma das opcoes

====== Menu de Livros ======
 1 - Adicionar Novo Livro    -> Inicia o cadastro de um Livro
 2 - Imprimir Livro (Titulo) -> Pede um Titulo, entao imprime se encontrar
 3 - Imprimir Livro (ID)     -> Pede um ID, entao imprime se encontrar
 4 - Imprimir Lista Completa -> Imprime todos os Livros cadastrados
 5 - Quantidade de Livros    -> Imprime a quantidade de Livros e de Exemplares
-1 - Voltar                  -> Volta para o Menu Inicial

====== Menu de Usuarios ======
 1 - Adicionar Novo Usuario  -> Inicia o cadastro de um Usuario
 2 - Imprimir Lista          -> Imprime todos Usuarios cadastrados
-1 - Voltar                  -> Volta para o Menu Inicial

====== Menu de Emprestimos ======
 1 - Criar Novo Emprestimo   -> Inicia o cadastro de um Emprestimo
 2 - Devolver um Livro       -> Pede os IDs do Usuario e Livro para realizar uma devolucao
 3 - Imprimir Lista          -> imprime todos Emprestimos
-1 - Voltar                  -> Volta para o Menu Inicial

====== Outros ======
 1 - Iniciar Arquivos  -> IMPORTANTE! Reinicia os arquivos do Programa
 2 - Carregar Arquivo  -> Pede o Diretorio de uma arquivo txt para Ler
 3 - Extras            -> Abre um Menu de Coisas desnecesarias 
-1 - Voltar            -> Volta para o Menu inicial
